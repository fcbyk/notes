# 域名由来
fcbyk 是 非常不愉快 的每个字的首拼音组合<br>
非常不愉快这个名字源于昵称重复
> 很久很久以前，我在某APP设置昵称 <br>
> 具体的 APP/游戏 和 当时想设置的昵称忘了，以网易云音乐APP和小明昵称举例

我尝试了 小明，小小明，明小，名字重复，id重复，为什么，我趣，????.....
